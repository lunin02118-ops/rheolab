# Archived Site Variants

Этот каталог хранит старые публичные варианты сайта, которые больше не участвуют
в продовой сборке.

С 12 апреля 2026 канонической считается версия, которая отображается на
`https://rheolab.site/`:

- главная страница публикуется только по маршруту `/`
- публичная FAQ публикуется только по маршруту `/faq`
- маршруты `v2`, `v3` и `v3-faq` убраны из prod-сборки

Файлы ниже сохранены только как архив исходников для возможного разбора истории
контента и дизайна.
