Калибровка приборов

RheoLab Enterprise предоставляет инструменты для автоматического анализа калибровочных данных реометров.

Поддерживаемые приборы

  • Chandler 5550 — Excel файлы с листом “Калибровка”
  • BSL R1 — Excel файлы с листом “Calibration”
  • Chandler CSV — CSV файлы калибровки

Процесс калибровки

При загрузке файла с калибровочными данными в приложении автоматически появляется вкладка “Калибровка”.

Метрики качества

Система автоматически рассчитывает ключевые показатели состояния прибора:

ПараметрНормаОписание
R² (Линейность)> 0.9999Точность линейной регрессии пружины
Гистерезис< 10 dyne/cm²Трение в системе (разница нагрузка/разгрузка)
StDev< 5 dyne/cm²Стандартное отклонение от идеальной прямой

Статусы

  • ✅ PASS — Прибор исправен, калибровка в норме.
  • ❌ FAIL — Требуется обслуживание или повторная калибровка.

Графики

  • Калибровочная кривая — Зависимость напряжения от угла отклонения.
  • Гистерезис — График разницы показаний при нагрузке и разгрузке.
