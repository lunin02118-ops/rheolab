## Быстрый старт

Если у вас чистый VPS с Ubuntu 24.04, вы можете использовать скрипт автоустановки:

```bash
# Загрузите скрипт установки
scp install.sh root@your-server-ip:/root/

# Подключитесь к серверу
ssh root@your-server-ip

# Запустите скрипт
chmod +x install.sh
./install.sh
```

Подробнее см. в [Руководстве по установке](INSTALLATION_RU.md).
