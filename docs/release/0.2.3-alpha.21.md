# RheoLab Enterprise 0.2.3-alpha.21

Published: 2026-06-12
Channel: alpha

## Scope

`0.2.3-alpha.21` publishes the new `О программе` dialog: support contacts,
MAX link and QR, training-video link and QR, plus the existing license
activation flow on a separate tab.
