# RheoLab Enterprise 0.2.3-alpha.22

Published: 2026-06-12
Channel: alpha

## Scope

`0.2.3-alpha.22` is a licensing hotfix for fresh desktop installs: when no
local license is present, the app now enters the local 30-day demo/trial mode
instead of blocking startup with `Лицензия не активирована`.

## Known Notes

- The server-side `register_demo.php` endpoint remains removed. This hotfix
  restores the desktop-local 30-day trial path only; a server anti-reset anchor
  should be handled as a separate licensing design task if needed.
- The release-gate teardown printed a Windows `EPERM` cleanup warning for an
  isolated WebView2 temp directory after the test process exited; the gate
  itself passed and returned exit 0.
