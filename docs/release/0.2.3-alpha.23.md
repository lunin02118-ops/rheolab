# RheoLab Enterprise 0.2.3-alpha.23

Published: 2026-06-12
Channels: alpha, beta, stable

## Scope

`0.2.3-alpha.23` refines the new `О программе` dialog after UI review:
the dialog now has three explicit tabs, the old duplicate About block is
removed from settings, and the contact email is consolidated to
`info@rheolab.site`.
