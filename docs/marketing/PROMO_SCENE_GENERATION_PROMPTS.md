# Промпты Для Генерации Сцен Промо-Ролика

Актуальный рабочий пакет теперь собран в одной папке:

[PROMO_SCENE_PACK_RU.md](video-scripts/promo-generated-photo-frames/PROMO_SCENE_PACK_RU.md)

В нём уже есть:

- все 10 опорных кадров;
- контактный лист;
- таймкоды;
- титры;
- текст озвучки;
- русский промпт для генерации видео по каждой сцене;
- общий негативный промпт;
- отдельный строгий негативный промпт для первой сцены с реометрами.

## Главное Правило

Если в сцене есть экран RheoLab, интерфейс программы нельзя перерисовывать.

Нельзя менять:

- вкладки;
- графики;
- кнопки;
- подписи;
- логотип;
- содержимое таблиц;
- состояние экрана.

Движение добавляется только в реальный мир вокруг интерфейса: камера, свет, отражения, бумага, позы людей, приборы, глубина резкости.

## Негативный Промпт

```text
Не перерисовывать интерфейс программы. Не придумывать новые кнопки, графики, меню, подписи или логотипы. Не добавлять читаемый фальшивый текст. Не менять содержимое экрана RheoLab. Не добавлять галлюцинированный интерфейс, стоковые водяные знаки, лишние бренды, искаженные руки или нечитаемые подписи графиков.
```
