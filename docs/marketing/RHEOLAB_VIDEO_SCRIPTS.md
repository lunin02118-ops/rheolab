# Сценарии видео RheoLab Enterprise

Оглавление сценарного пакета для сайта и промо-материалов.

## Документы

| Видео | Документ | Назначение |
|---|---|---|
| Общие правила | [PRODUCTION_GUIDELINES.md](PRODUCTION_GUIDELINES.md) | Требования к съёмке, монтажу и формулировкам |
| Настройка программы | [SETUP.md](SETUP.md) | Первый запуск, базовые параметры, лицензия, первый файл |
| Вкладка «Анализ» | [ANALYSIS.md](ANALYSIS.md) | Загрузка файла, график, сырые данные, рецептура, вода, отчёт и сохранение |
| Работа с базой данных | [LIBRARY.md](LIBRARY.md) | Библиотека экспериментов, фильтры, поиск, реагенты |
| Вкладка «Сравнение» | [COMPARE.md](COMPARE.md) | Добавление опытов, общие оси, легенда, отчёт по сравнению |
| Выгрузка отчётов | [REPORTS.md](REPORTS.md) | PDF / Excel, язык, разделы, сохранение настроек |
| Первое знакомство | [FIRST_LOOK_OVERVIEW.md](FIRST_LOOK_OVERVIEW.md) | Обзор вкладок Анализ, Библиотека, Сравнение и Настройки |
| Промо-ролик | [PROMO.md](PROMO.md) | Польза для полевых лабораторий, ГРП/бурения, НИИ и производителей химии |
| Опорные кадры промо | [PROMO_REFERENCE_FRAMES.md](PROMO_REFERENCE_FRAMES.md) | Реальные UI-кадры и b-roll prompts для генерации видео |
| Prompts для сцен | [PROMO_SCENE_GENERATION_PROMPTS.md](PROMO_SCENE_GENERATION_PROMPTS.md) | Подсказки для генерации сцен по готовым reference frames |

## Карта роликов на сайте

| ID | Заголовок | Назначение |
|---|---|---|
| `setup` | Настройка программы | Первый запуск, основные параметры и короткий маршрут до первого результата |
| `analysis` | Вкладка «Анализ» | Загрузка файла, проверка данных, расчёты, отчёт и сохранение в базу |
| `library` | Работа с базой данных | Как устроена библиотека опытов, поиск и повторное открытие результатов |
| `compare` | Вкладка «Сравнение» | Сравнение нескольких опытов на единых осях для наглядного анализа рецептур |
| `reports` | Выгрузка отчетов | Как получить PDF или Excel с графиком и ключевыми значениями |
| `first-look` | Первое знакомство | Краткий обзор основных вкладок программы и рабочих сценариев |
| `promo` | Промо-ролик | От сырого файла к единой методике, базе экспериментов, сравнению и отчёту |
